So what exactly is HTML? HTML provides structure to the content appearing on the website,such as images, text, or videos.Right-click on any page on the internet,choose "Inspect,"and you will see HTML in a panel of your screen.

HTML stands for Hyper Text MarkUp Language:

  • A markup language is a computer language that defines the structure and presentation of a raw text.

  • In HTML, the computer can interpret raw texts that is wrapped in HTML elements.

  • Hyper Text is a text displayed on a computer or device that provdes access to other texts through links, also known as hyperlinks. You probably clicked on several hyperlinks on your way to this codingMadeEasy course.

Learning HTML is the first step in creating websites,but even a little knowledge can help you inject code snippets into newspapers, blog, or websites templates. As you continue learning, you will be able to layer your HTML with CSS and JavaScript to create visual compelling websites. But for now, we are going to focus on how to add and modify basic content on a page,like texts,images and videos.Don't worry if the website looks ugly-we are just getting started.
